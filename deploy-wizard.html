🚀 LabEscape 自动部署助手

按照以下步骤完成游戏上线部署

第一步：推送代码到 GitHub 待完成

仓库地址：

https://github.com/yangdongchen66-boop/labescape.git

方法 A：使用 GitHub Desktop（推荐）

  1. 打开 GitHub Desktop
  2. File → Add Local Repository → 选择 E:\chatAI
  3. 点击 "Push origin" 按钮

方法 B：命令行

cd E:\chatAI git remote set-url origin https://github.com/yangdongchen66-boop/labescape.git git push origin master
⚠️ 注意： 如果遇到网络错误，请使用 GitHub Desktop 或直接访问 GitHub 网页上传文件。
查看 GitHub 仓库 下载 GitHub Desktop

第二步：连接 Railway 待完成

  1. 访问 Railway 并登录
  2. 创建新项目，选择 "Deploy from GitHub repo"
  3. 授权并选择 labescape 仓库
  4. Settings → Root Directory 设置为 backend
  5. 添加环境变量：
    • DEEPSEEK_API_KEY = your_api_key
    • SERVER_PORT = 8080
    • JAVA_VERSION = 21
  6. 等待自动部署完成（约 3-5 分钟）
  7. Settings → Domains → Generate Domain，复制生成的 URL
打开 Railway Dashboard

第三步：更新 Vercel 配置 待完成

  1. 访问 Vercel Dashboard
  2. 选择 frontend 项目
  3. Settings → Environment Variables
  4. 编辑 VITE_API_BASE_URL
  5. 设置为你的 Railway URL
  6. 保存并重新部署（Redeploy）
打开 Vercel Dashboard 查看前端预览

✅ 完成！测试游戏

访问游戏：

https://frontend-4a29rb70r-yangdongchens-projects.vercel.app

测试清单：

  • 页面正常加载
  • 能看到开场剧情
  • HP/MP/金币显示正常
  • 点击行动选项能触发骰子动画
  • 濒临崩溃时出现红色边框警告
  • NPC 对话显示态度表情

验证后端健康检查：

curl https://your-railway-url.app/api/game/health
