🎛️ Railway 环境变量配置

只需 3 分钟，轻松搞定后端配置！

0% 完成

1打开 Railway Dashboard

点击下面的按钮打开 Railway 控制台：

🚀 打开 Railway Dashboard

👉 登录后，找到你的 labescape 项目并点击进入

2切换到 Variables 标签

在项目页面顶部导航栏，点击 "Variables" 标签

📍 位置：Overview | Deployments | Variables | Settings

3添加 3 个环境变量

点击 "New Variable" 按钮，依次添加以下变量：

DEEPSEEK_API_KEY

值：

sk_xxxxxxxxxxxxxxxxxxxx

⚠️ 替换为你的真实 DeepSeek API Key

SERVER_PORT

值：

8080
JAVA_VERSION

值：

21
⚠️ 注意： 确保变量名完全正确（大写字母），不要有拼写错误！
🎉 完成！Railway 会自动重新部署，等待 2-3 分钟即可

✅ 验证清单

  • 成功登录 Railway Dashboard
  • 找到了 labescape 项目
  • 点击了 Variables 标签
  • 添加了 DEEPSEEK_API_KEY
  • 添加了 SERVER_PORT = 8080
  • 添加了 JAVA_VERSION = 21
  • 看到所有 3 个变量显示在列表中

🎯 下一步

  1. 等待 Railway 自动重新部署（2-3 分钟）
  2. 在 Deployments 标签查看部署状态
  3. 部署成功后，获取生成的域名 URL
  4. 将该 URL 配置到 Vercel 的 VITE_API_BASE_URL
VITE_API_BASE_URL=https://your-app.up.railway.app
