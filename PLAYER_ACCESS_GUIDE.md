### 方式二：自定义域名（更专业）

如果您有自己的域名，可以配置：

#### 前端自定义域名

**步骤：**

1. **Vercel 设置**
   - Vercel Dashboard → Settings → Domains
   - 添加自定义域名（如：play.labescape.com）
   - 按照提示配置 DNS

2. **DNS 配置**
   ```
   Type: CNAME
   Name: play
   Value: cname.vercel-dns.com
   ```

3. **等待生效**
   - DNS 传播需要 5-30 分钟
   - SSL 证书自动签发

#### 后端自定义域名

同样可以在 Railway 配置：
- Railway Dashboard → Settings → Domains
- 添加 API 域名（如：api.labescape.com）

#### 更新前端配置

记得在 Vercel 更新环境变量：
```
VITE_API_BASE_URL=https://api.labescape.com
```

然后重新部署前端。

## 🆘 常见问题 FAQ

### Q1: 玩家说打不开链接？

**可能原因：**
- 网络问题（国内访问可能需要加速器）
- 浏览器版本过旧
- 本地网络限制

**解决方案：**
1. 确认玩家使用现代浏览器
2. 尝试清除浏览器缓存
3. 更换网络环境
4. 使用手机流量测试

### Q2: 游戏加载很慢？

**可能原因：**
- 首次访问需要加载资源
- 网络连接慢
- 后端响应延迟

**解决方案：**
1. 耐心等待首次加载（约 5-10 秒）
2. 检查网络连接
3. 刷新页面重试

### Q3: 玩家进度丢失？

**说明：**
- 当前版本使用内存存储
- 关闭浏览器会丢失进度
- 同一设备刷新页面会保留

**未来改进：**
- 添加用户登录系统
- 使用数据库持久化存储
- 支持云存档

### Q4: 可以支持多少玩家同时在线？

**理论值：**
- Vercel：无限并发（CDN 分发）
- Railway：受限于 API 调用速度
- 建议：50-100 人同时在线

**瓶颈：**
- DeepSeek API 调用频率
- Railway 服务器资源
- 数据库连接数

### Q5: 如何盈利？

**可能的商业模式：**
- 捐赠支持（爱发电）
- 高级功能订阅
- 自定义剧情包
- 去除广告版本
